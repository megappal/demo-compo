# DemoMegappal Project

This project hosts personalized music demos created using AI tools, including music generated by SUNO AI and images created with AI-based generators.

## About

- The website showcases a collection of personalized songs, each accompanied by themed artwork and descriptive information cards.
- The interactive demo player presents one song at a time with navigation controls.
